# CHRONYX — A Quiet Space for Your Life

A private, calm system to hold your life with continuity. Tasks, learning, money, and memories — all in one quiet, focused space.

## About

CHRONYX by CROPXON is a comprehensive life management platform designed to help you organize every aspect of your life in one unified, peaceful interface.

## Features

- **Task Management** - Keep track of your daily tasks and todos
- **Study Tracking** - Monitor your learning progress with built-in timers and analytics
- **Financial Management** - Track expenses, income, loans, and insurance
- **Memory Vault** - Store and organize your precious memories
- **Documents** - Secure document storage and management
- **Lifespan Visualization** - Unique perspective on time and life goals
- **IDEAs BRAINSTORMING** - Unique ideas for future scope
- **PERSONAL DIGITAL DIARY** - A Digital Diary For Personal Use


## Technology Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Shadcn UI
- Framer Motion

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```
